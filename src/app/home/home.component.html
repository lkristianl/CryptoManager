Bienvenidos a CryptoManager

CryptoManager es una aplicación web donde puedes gestionar todas tus criptomonedas y exchanges desde un único sitio, sin tener que estar abriendo varias ventanas de tu navegador para conectarse a varios exchanges a la vez, gracias a las APIs con las que trabaja CryptoManager.

Para ello solo necesitas registrarte con un correo electrónico y una contraseña, con los que podrás acceder a tu cuenta. Luego deberás configurar tu cuenta para establecer la conexión con el exchange que deseas utilizar (Configurar API de un exchange en CryptoManager), y así poder gestionar tus criptomonedas de una manera más ágil y sencilla.

Los servicios que ofrece CryptoManager son:

1. Gestionar órdenes de compra y venta

CryptoManager facilita la gestión de las distintas órdenes de compra y venta que abre, cancela o tiene pendientes, ya sea en un exchange específico, como en todos aquellos que estén relacionados con su cuenta de CryptoManager.

2. Gestionar balance

CryptoManager muestra a los usuarios tanto el balance para realizar operaciones de compra como el valor total de sus criptomonedas, en cada exchange y en global.

3. Gestionar depósitos y retiros de monedas FIAT

Con CryptoManager, los usuarios podran realizar depósitos y retiros de monedas FIAT (€, $, etc) desde y hacia sus cuentas bancarias o tarjetas.

4. Gestionar tranferencias entre carteras

CryptoManager agiliza las transferencias entre las distintas carteras que tiene un usuario en cada exchange. Gracias a una interfaz de usuario más sencilla y precisa.

5. Mostrar historiales

Con CryptoManager los usuarios podrán ver sus historiales de órdenes de compra y venta, de depósitos y retiros, y de transferencias entre carteras.

6. Mostrar información de un par de activos

Cuando un usuario va a realizar una orden de compra o venta, CryptoManager le facilitará toda la información relacionada con el par de activos, donde el usuario quiere realizar dicha orden.
