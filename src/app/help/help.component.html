CONFIGURAR UNA CLAVE API EN CRYPTOMANAGER

A la hora de configurar una clave API en CryptoManager de un exchange en concreto debemos realizar los siguientes pasos:

Paso 1:
Tenemos que iniciar sesión en nuestra cuenta de CryptoManager.
Paso 2:
A continuación, nos dirigimos a la configuración de nuestro perfil y seleccionamos la opción "Añadir una API".
Paso 3:
Rellenamos los datos del formulario (El exchange al que pertenece la API, la clave pública y privada [REVISARRRRRR], etc).
Paso 4:
Cuando hayamos rellenado todos los campos obligatorios, hacemos click en el botón "Añadir API".
Paso 5:
Para comprobar que la API se ha añadido correctamente, nos dirigiremos al panel de control, donde nos debería de aparecer el nuevo exchange.
